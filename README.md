## 🏏 Project Overview

This project is a simple end-to-end **data engineering pipeline** built using cricket match data from an API. It covers the full lifecycle of data — from extraction 🌐, to transformation 🔧, and finally loading into a database 🗄️ where it can be queried and analysed 📊.
